
## Tentang Aplikasi

Aplikasi Mankeu ini adalah aplikasi keuangan sederhana yang berisi kalkulator perhitungan time value of money. Aplikasi ini dibuat guna untuk memenuhi tugas akhir mata kuliah Manajemen Keuangan.
